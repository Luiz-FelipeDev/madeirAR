# mandeirAR

## Visão Geral do Projeto

O **mandeirAR** é um projeto inovador que visa revolucionar a forma como os usuários interagem com móveis de madeira personalizados, utilizando a tecnologia de Realidade Aumentada (AR). Através de uma interface intuitiva, os usuários podem configurar e visualizar móveis em seus próprios ambientes, garantindo uma experiência de compra e personalização sem precedentes.

## Funcionalidades Principais

### 1. Configuração de Móveis Personalizados

O aplicativo oferece uma interface rica e fácil de usar para a personalização de móveis. Os usuários podem escolher entre diversas opções para criar o móvel ideal:

*   **Bases:** Seleção de diferentes tipos de bases, como Cavalete Ajustável, Escultura e Tripla.
*   **Formato do Tampo:** Opções de tampos Retangulares e Redondos.
*   **Acabamentos:** Uma vasta gama de acabamentos de madeira, permitindo ao usuário escolher a textura e a cor que melhor se adapta ao seu ambiente.
*   **Decorações:** Adição de elementos decorativos para complementar o design do móvel.

### 2. Visualização em Realidade Aumentada (AR)

A principal característica do mandeirAR é a capacidade de visualizar o móvel configurado em tempo real no ambiente do usuário. Isso permite:

*   **Imersão:** Uma experiência imersiva que ajuda o usuário a ter uma ideia precisa de como o móvel ficará em seu espaço.
*   **Ajuste de Escala:** Ferramentas para ajustar a escala do móvel, garantindo que ele se encaixe perfeitamente no local desejado.
*   **Posicionamento:** Facilidade para posicionar e rotacionar o móvel virtualmente, explorando diferentes arranjos.

### 3. Interface Intuitiva e Detalhada

A interface do usuário é projetada para ser clara e funcional, com controles deslizantes para ajustes finos e botões de seleção para opções pré-definidas. Detalhes como a medição em metros (ex: 1.0m) são exibidos para auxiliar na tomada de decisão.

## Como Funciona

1.  **Personalize:** Utilize a interface de configuração para escolher as características do seu móvel.
2.  **Visualize:** Ative o modo AR para ver o móvel em seu ambiente real.
3.  **Ajuste:** Faça os ajustes necessários de tamanho e posição.
4.  **Solicite:** Envie sua configuração para solicitar um orçamento ou realizar a compra.

## Tecnologias Envolvidas

Embora as tecnologias exatas não sejam explicitamente visíveis nas imagens, é evidente que o projeto utiliza:

*   **Realidade Aumentada (AR):** Para a sobreposição de modelos 3D no ambiente real.
*   **Renderização 3D:** Para exibir os modelos de móveis com diferentes texturas e acabamentos.
*   **Interface de Usuário (UI):** Componentes interativos para seleção e ajuste de opções.
